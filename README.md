# Python学生信息管理系统 python909

## 项目概述

本项目是一款基于Python的学生信息管理系统，集成了登录注册、数据总览、学生信息管理、学生成绩管理、教师信息管理、考试信息管理、通知管理等主要功能模块。

## 技术栈

- **后端**: Python + Flask
- **前端**: HTML + CSS + JavaScript + jQuery
- **数据库**: MySQL
- **开发工具**: PyCharm、Navicat

## 系统角色

- 管理员：负责系统整体管理与通知发布
- 教师：负责个人信息维护、成绩录入与考试信息管理
- 学生：负责个人信息查看、成绩查询等

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0及以上版本

## 部署步骤

1. 环境配置：安装Python 3.0以上版本、MySQL 8.0以上版本、PyCharm和Navicat工具。
2. 数据库搭建：根据提供的SQL脚本，搭建项目所需数据库结构。
3. 代码部署：将代码部署到支持Flask的服务器上。
4. 配置运行：配置应用设置，确保应用能正确连接数据库，并进行调试。
